## 项目简介(Profile)
当初做这个项目，完全是因为想出去旅游，却苦于一大堆的旅游攻略，不知道怎么选择，又不想跟着旅游团走，所以get了这个“伴旅”的idea。

这样的话，每个用户都能成为一名私人导游，在自己熟悉的地方，带着一些不熟悉环境的游客旅游。而游客可以支付一定的佣金去雇佣一位合适的导游。只要双方达成协议，就能让双方获得利益。
